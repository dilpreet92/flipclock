flipclock
=========

flipclock is a [**flip clock**][1] [**jQuery**][2] plugin with [**CSS3 animations**][3] for modern browsers.
Also, you can use it as a [**Zepto**][4] plugin.

USAGE
-------
clock

    $('#container').flipclock();
    
full date

    $('#container').flipclock('date');
    
countdown

    $('#container').flipclock('2014 08 18 12:00:00');
